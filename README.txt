Dans le projet : BanqueService-0.0.1-SNAPSHOT : configurer la connexion à la base de donnée (nom de base, username et password) dans BanqueService-0.0.1-SNAPSHOT/src/hibernate.cfg.xml !

Le projet BanqueService-0.0.1-SNAPSHOT doit être sur un serveur sur le port 8080.
Le projet Gateway_Banque_Service_SOAP doit être sur un serveur sur le port 10080.

Les serveurs et les WebServices doivent être lancé avant de lancer les clients.

Base de donnée utilisé : ma_base.sql
